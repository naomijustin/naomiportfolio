Tribute: The Art of Nigirizushi
Introduction
Aim is to experiment in creation of web illustrations purely using SVG Paths, Clipping Paths and Linear Gradients in the form of a tribute page.

Page Mockup
Mockup after a few variations of style, deciding on layout of page and design of code illustrations.

Sketch vs Code Illustration

Colour
Colourized these illustrations with linear gradients to create the illusion of a 3D shading. Selectively using accented colours that were bold against the grey scale background to make these images stand out.

• Get Waves
